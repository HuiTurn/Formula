## 基础

### 功率公式

**公式ID**: `middle_phy_006`

**公式**: `P = W/t = Fv`

**LaTeX**: `P = \frac{W}{t} = Fv`

**变量说明**:

- **P**: 功率 (单位: W (瓦特))
- **W**: 功 (单位: J (焦耳))
- **t**: 时间 (单位: s (秒))
- **F**: 力 (单位: N (牛顿))
- **v**: 速度 (单位: m/s (米每秒))

**适用条件**: t > 0

**备注**: 功率等于功除以时间，或等于力乘以速度

**推导**: 功率定义

**标签**: 常用公式, 必背公式
