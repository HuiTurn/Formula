## 描述统计

### Z分数（标准化）

**公式ID**: `stat_009`

**公式**: `z = (x - μ) / σ`

**LaTeX**: `z = \frac{x - \mu}{\sigma}`

**变量说明**:

- **z**: Z分数
- **x**: 原始数据值
- **μ**: 均值
- **σ**: 标准差

**适用条件**: σ > 0

**备注**: 将数据标准化为标准正态分布，z=0表示等于均值，z=±1表示距离均值1个标准差

**推导**: 统计定义

**标签**: 常用公式
