## 力学

### 牛顿第二定律

**公式ID**: `physics_001`

**公式**: `F = ma`

**LaTeX**: `F = ma`

**变量说明**:

- **F**: 物体所受的合外力 (单位: N (牛顿))
- **m**: 物体的质量 (单位: kg (千克))
- **a**: 物体的加速度 (单位: m/s² (米每二次方秒))

**适用条件**: 适用于宏观低速物体

**备注**: 这是经典力学的基础公式

**推导**: 通过实验总结得出

**标签**: 常用公式, 必背公式

### 动能公式

**公式ID**: `physics_002`

**公式**: `Ek = ½mv²`

**LaTeX**: `E_k = \frac{1}{2}mv^2`

**变量说明**:

- **Ek**: 物体的动能 (单位: J (焦耳))
- **m**: 物体的质量 (单位: kg (千克))
- **v**: 物体的速度 (单位: m/s (米每秒))

**适用条件**: 适用于宏观低速物体

**备注**: 动能与速度的平方成正比

**推导**: 由功的定义和牛顿第二定律推导

**标签**: 常用公式

### 万有引力定律

**公式ID**: `physics_003`

**公式**: `F = G(m₁m₂)/r²`

**LaTeX**: `F = G\frac{m_1m_2}{r^2}`

**变量说明**:

- **F**: 万有引力 (单位: N (牛顿))
- **G**: 万有引力常数 (单位: 6.67×10⁻¹¹ N·m²/kg²)
- **m₁**: 物体1的质量 (单位: kg (千克))
- **m₂**: 物体2的质量 (单位: kg (千克))
- **r**: 两物体间的距离 (单位: m (米))

**适用条件**: 适用于质点或均匀球体

**备注**: G 是万有引力常数，约为 6.67×10⁻¹¹ N·m²/kg²

**推导**: 由开普勒定律和牛顿运动定律推导

**标签**: 常用公式

### 自由落体运动

**公式ID**: `high_phy_004`

**公式**: `v = gt, h = (1/2)gt², v² = 2gh`

**LaTeX**: `v = gt, \quad h = \frac{1}{2}gt^2, \quad v^2 = 2gh`

**变量说明**:

- **v**: 速度 (单位: m/s)
- **h**: 下落高度 (单位: m)
- **g**: 重力加速度 (单位: 9.8 m/s²)
- **t**: 时间 (单位: s)

**适用条件**: 初速度为零，只受重力作用

**备注**: 自由落体是初速度为零的匀加速直线运动

**推导**: 由匀变速直线运动公式，令 v₀ = 0, a = g

**标签**: 常用公式, 必背公式

### 平抛运动

**公式ID**: `high_phy_005`

**公式**: `x = v₀t, y = (1/2)gt², vₓ = v₀, vᵧ = gt`

**LaTeX**: `x = v_0t, \quad y = \frac{1}{2}gt^2, \quad v_x = v_0, \quad v_y = gt`

**变量说明**:

- **x**: 水平位移 (单位: m)
- **y**: 竖直位移 (单位: m)
- **v₀**: 初速度 (单位: m/s)
- **vₓ**: 水平速度 (单位: m/s)
- **vᵧ**: 竖直速度 (单位: m/s)
- **t**: 时间 (单位: s)

**适用条件**: 初速度水平，只受重力作用

**备注**: 平抛运动可以分解为水平方向的匀速直线运动和竖直方向的自由落体运动

**推导**: 运动的合成与分解

**标签**: 常用公式, 必背公式

### 向心力公式

**LaTeX**: `F = \frac{mv^2}{r} = m\omega^2r = \frac{4\pi^2mr}{T^2}`

**变量说明**:

**备注**: 向心力指向圆心，提供物体做圆周运动所需的向心加速度

**推导**: 由牛顿第二定律和圆周运动加速度推导

**标签**: 常用公式, 必背公式

**备注**: 行星公转周期的平方与轨道半长轴的立方成正比

### 库仑定律

**公式ID**: `high_phy_014`

**公式**: `F = k(q₁q₂)/r²`

**LaTeX**: `F = k\frac{q_1q_2}{r^2}`

**变量说明**:

- **F**: 静电力 (单位: N)
- **k**: 静电力常量 (单位: 9×10⁹ N·m²/C²)
- **q₁, q₂**: 电荷量 (单位: C)
- **r**: 距离 (单位: m)

**适用条件**: 点电荷，真空或空气中

**备注**: 真空中两个点电荷之间的相互作用力与它们的电荷量的乘积成正比，与它们之间距离的平方成反比

**推导**: 库仑定律

**标签**: 常用公式, 必背公式
