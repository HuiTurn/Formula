## 描述统计

### 平均值公式

**公式ID**: `stat_001`

**公式**: `x̄ = (x₁ + x₂ + ... + xₙ) / n`

$$
\bar{x} = \frac{x_1 + x_2 + \cdots + x_n}{n}
$$

**变量说明**:

- **x̄**: 平均值
- **x₁, x₂, ..., xₙ**: 各个数据值
- **n**: 数据个数

**适用条件**: n > 0

**备注**: 所有数据值的和除以数据个数

**推导**: 统计定义

**标签**: 常用公式
